 O que o projeto faz:
Este projeto consiste em um sistema inteligente de monitoramento para controle operacional básico de uma missão espacial experimental. Desenvolvido em Python, o sistema gera telemetrias simuladas em tempo real de temperatura, níveis de bateria e estabilidade de comunicação. Utilizando uma camada lógica integrada a estruturas condicionais, a plataforma gera alertas de segurança automáticos. Complementarmente, o modelo de linguagem **Llama 3.2 1B (via Ollama)** atua como engenheiro chefe virtual, interpretando dados críticos e gerando insights e diagnósticos acionáveis para mitigação de riscos na operação aeroespacial.
